Objectif

Quel est l'objectif de ce site ?

Ce site a pour objectif de simplifier la consultation des mesures des différents capteurs et voir le trajet et la position des objets dans la pièce.

Vous trouverez sur ce site:

La page "Accueil" qui affiche l'objectif de notre site ainsi que ce qu'ont pourra trouver dessus.

La page "Consultation" son pour voir les différentes positions de l'objet dans la salle via les capteurs de sons.

La page "Consultation" ultrason pour voir les différentes positions de l'objet, de la personne dans la salle via les capteurs ultrasons.

La page "Connexion" qui permettra de se connecter en tant qu'administrateur et avoir des droits supplémentaires.

La page "Gestion de Projet" qui affiche le GANTT final, la synthèse personnelle de chaque membre, etc.

Enfin, la page Mentions Légales affichant les caractéristiques du site web.
